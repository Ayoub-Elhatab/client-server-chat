# Client-Server Chat Application

A simple TCP-based chat application in Java for communication between a server and client.

## Description

Basic socket programming implementation where a server listens for client connections on port 1234. The client sends messages and receives acknowledgment responses from the server.

## How to Run

### Prerequisites
- JDK 8 or higher

### Steps

1. **Compile the files**
2. **Start the Server first** - Run Server.java
3. **Start the Client** - Run Client.java in a new terminal
4. **Send messages** - Type in the client terminal and press Enter
5. **Disconnect** - Type "EXIT" to close the connection

## Features

- TCP socket communication
- Real-time message exchange
- Handles one client at a time
- Type "EXIT" to disconnect

## Configuration

- **Port**: 1234
- **Host**: localhost

Both values can be modified in the source files.
